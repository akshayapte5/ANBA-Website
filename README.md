# ANBA After Wear — Premium Streetwear Brand

This is the official landing page for **ANBA After Wear**, a premium minimalist streetwear brand.

## Features
- **Zero-Build CMS**: Change text/images in `content/` files without touching HTML.
- **Scroll Reveal**: High-end entrance animations.
- **Parallax Imagery**: Immersive "window-pane" photo effects.
- **Responsive Navigation**: Mobile-optimized sidebar and sticky glass navbar.

## Tech Stack
- HTML5 / CSS3 / Vanilla JS
- Tailwind CSS (CDN)
- Feather Icons
